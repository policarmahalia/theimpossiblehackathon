## How to Run
1. Download and install [Godot Engine 4.6](https://godotengine.org/)
2. Clone this repository
3. Open Godot → Import Project → select the project folder
4. Press F5 or hit the Play button to run

## About the Game
Bun Bun the Dum Dum is a bunny with low health who needs to eat. 
Help Bun Bun solve increasingly unhinged quiz puzzles to unlock food 
and restore health — but don't make the AI helper Eyestein too angry 
or face the consequences.
